# fz_task_1

Description:
This is a mini pos desktop app.
It uses Hive for local storage with 4 models for users, items, checkout and checkout's items.
It has a login page with session management.
After logging in, the user is routed to the Home Page which has 3 buttons: Items management, checkout and bills.
The Items management page displays items and allows Filter, Create, Update and Delete actions.
The Checkout page page also displays items and allows Filter action, and it allows for cart management.
The Bills page displays all bills and allows a filter action.

Setup:
Just pull the project from Github: https://github.com/ZaYa456/fz_task_1_mini_pos, then run.
